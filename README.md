# Personal Geo-Journal

A personal interactive web map to document and visualize places I have traveled to or wish to visit.

## Purpose
This project is a tech-savvy travel enthusiast's tool for mapping and journaling travel experiences. It combines interests in mapping, travel, and new technologies by building a functional interactive map from scratch while adhering to modern development practices.

## Technologies Used
- HTML5, CSS3, JavaScript (ES6+)
- [Leaflet.js](https://leafletjs.com/) - an open-source JavaScript library for interactive maps
- GitHub Pages for hosting

## Live Site
Once deployed, the live site will be available at: [https://tomaseshithari.github.io/personal-geo-journal/](https://tomaseshithari.github.io/personal-geo-journal/)

## Features (Planned)
1. Basic interactive map centered on a location of choice
2. Ability to add markers for visited destinations
3. Journal entries linked to markers
4. Filtering by trip type, date, etc.
5. Responsive design for mobile and desktop
